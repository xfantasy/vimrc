还可以进一步指定上下文:
                                                        *help-context*
                          类别                      前缀     例子   ~
                      普通模式命令                (无)      :help x
                      可视模式命令                 v_        :help v_u
                      插入模式命令                 i_        :help i_<Esc>
                      命令行模式命令              :         :help :quit
                      命令行编辑            c_        :help c_<Del>
                      Vim 命令参数           -         :help -r
                      选项                     '         :help 'textwidth'
         查找帮助:  键入 ":help word"，接着键入 CTRL-D 可以看到匹配 "word" 的
                    帮助主题。
